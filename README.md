# ✅ TaskManager

A full‑stack task management platform with real‑time features and a modern UI.

## ✨ Highlights
- RESTful backend with authentication & storage
- Real‑time updates (Socket.IO)
- React + Vite frontend
- Tailwind styling

## 🛠️ Tech Stack
**Backend**
- Node.js + Express
- MongoDB + Mongoose
- Firebase Admin
- Socket.IO

**Frontend**
- React 18
- Vite
- Tailwind CSS

## 🚀 Getting Started
### Backend
```bash
cd Backend
npm install
node app.js
```

### Frontend
```bash
cd Frontend/TaskManager
npm install
npm run dev
```

> Configure environment variables in `Backend/.env` and frontend `.env` as needed.

## 📁 Structure
```
Backend/
Frontend/TaskManager/
```
